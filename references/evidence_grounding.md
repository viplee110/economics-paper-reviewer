# Evidence Grounding

Economics review must be grounded in verified sources where possible.

## Preferred Sources

Use the following when tools are available:

- NBER
- SSRN
- RePEc / IDEAS
- journal pages
- publisher pages
- working paper series
- author pages
- OpenAlex
- Semantic Scholar
- user-provided PDFs
- arXiv only when relevant

## Evidence Status Labels

Use these labels:

- verified: source opened and the claim checked
- inferred: reasonable inference from available paper text, but not directly verified
- provisional: search unavailable, source unavailable, or claim not yet checked
- missing: required evidence not supplied

## Claims Requiring Evidence

These claims must be verified or marked provisional:

- closest paper
- benchmark paper
- method anchor
- identification anchor
- absorption threat
- style anchor
- structure anchor
- target-fit comparison
- data or code availability
- theorem/proof comparison to prior work

## No-Fabrication Rules

Do not invent:

- citations
- journal status
- referee evidence
- robustness results
- data availability
- code availability
- theorem statements
- proof validity
- experimental details
- institutional details

If evidence is missing, state the exact missing evidence and how it affects the review.
